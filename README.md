#  HabitTrackerCS

Egy egyszerű és hatékony asztali alkalmazás rossz szokások (alkohol, szerencsejáték stb.) követésére és leküzdésére.

##  Funkciók
- **Szokások kezelése:** Új szokások felvétele és törlése.
- **Időmérés:** Pontos visszaszámlálás az utolsó botlás óta.
- **Naplózás:** Megjegyzések fűzése az egyes eseményekhez.
- **Adattárolás:** SQLite adatbázis a helyi mentéshez.

##  Technológiai stack
- **Nyelv:** C# (.NET)
- **UI:** WPF (Windows Presentation Foundation)
- **Adatbázis:** SQLite

##  Telepítés és használat
1. Klónozd a repository-t: `git clone https://github.com/FELHASZNALONEV/HabitTrackerCS.git`
2. Nyisd meg a `.sln` fájlt Visual Studio 2022-ben.
3. Futtasd a projektet (F5).
